# XML/XSL 公文書変換ツール

XML ファイルおよび ZIP アーカイブに含まれる XML を、付随する XSL を用いて HTML に変換するためのツールです。  
日本年金機構などが提供する電子公文書形式に対応しています。

## 機能

- XML ファイルを HTML に変換
- ZIP ファイルを展開し、内部の XML を一括変換
- XML 内の `xml-stylesheet` から XSL を自動検出
- 変換後の HTML は入力ファイルと同じフォルダに出力
- 「機構からのお知らせ」欄の折り返し不具合を自動修正

## 使い方

1. `xml2html.exe` を任意の場所に配置します。
2. 変換したい XML または ZIP ファイルを exe にドラッグ＆ドロップします。
3. 同じフォルダに HTML ファイルが生成されます。

## 対応形式

- XML（XSLT1.0）
- ZIP（内部に XML/XSL が含まれるもの）

## 注意事項

- XSL は XML と同じフォルダ、または ZIP 内に存在する必要があります。
- 公文書のレイアウト仕様により、一部の項目は CSS を上書きして折り返し処理を行っています。
- GUI はありません。ドラッグ＆ドロップのみで動作します。

## 動作環境

- Windows 10 / 11
- Python 3.14 で開発し、PyInstaller により単一 exe 化

## ライセンス

本ツールは自由に利用・改変できます。
